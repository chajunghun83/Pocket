- "차입" → **"대출"** 용어 변경

### 4. 설정 페이지 대폭 개선

- **주식 기본 탭** 설정 (전체/한국/미국)
- **예산 목표** 설정 (+1만/+10만/+100만 버튼)
- **시작 페이지** 설정 (대시보드/가계부/부채/주식/설정)
- 카드 배치 변경: 계정|데이터 → 화면설정|예산목표 → 정보
- 다크모드 기본값으로 설정

### 5. 대시보드 페이지 전면 개편

## 🚀 다른 PC에서 시작하는 방법

### 1단계: 최신 코드 가져오기
```bash
cd C:\work\Pocket   # 또는 D:\work\Pocket
git pull
```

### 2단계: 패키지 설치 (처음 한 번만)
```bash
npm install
```

### 3단계: .env 파일 확인
`.env` 파일이 없으면 `env-template.txt` 참고해서 생성

### 4단계: 개발 서버 실행
```bash
npm run dev
```

### 5단계: 브라우저에서 확인
`http://localhost:3000` 접속
