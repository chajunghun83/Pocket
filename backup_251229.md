## 🚀 다른 PC에서 시작하는 방법

### 1단계: 최신 코드 가져오기
```bash
cd D:\work\Pocket
git pull
```

### 2단계: 패키지 설치 (처음 한 번만)
```bash
npm install
```

### 3단계: .env 파일 확인
`.env` 파일이 없으면 `env-template.txt`를 참고해서 생성:
```
VITE_SUPABASE_URL=https://gzxbckioutctwxbevqmk.supabase.co
VITE_SUPABASE_ANON_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
```

### 4단계: 개발 서버 실행
```bash
npm run dev
```

### 5단계: 브라우저에서 확인
`http://localhost:3000` 접속
